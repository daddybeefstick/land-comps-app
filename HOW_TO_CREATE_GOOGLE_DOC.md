# How to Create a Google Doc from Your Research Paper

## Option 1: Import the HTML File (Recommended)

1. **Open Google Drive** (drive.google.com) and sign in.

2. **Upload the file:**
   - Click **New** → **File upload**
   - Navigate to: `c:\Users\wjc32\land-comps-app\`
   - Select **William_Covert_Research_Paper_Google_Doc.html**
   - Click **Open**

3. **Convert to Google Docs:**
   - Find the uploaded file in your Drive
   - **Right-click** the file
   - Select **Open with** → **Google Docs**
   - Google will convert the HTML into an editable Google Doc

4. **Rename and organize:**
   - Use **File** → **Make a copy** or **Rename** to keep it as your working draft
   - Apply MLA formatting: double-space, Times New Roman 12pt, add page numbers (Insert → Page numbers)

## Option 2: Copy-Paste from the Markdown File

1. Open **William_Covert_Research_Paper_Revised.md** in any text editor.
2. Scroll to **Section 3: Rewritten Final Draft** (starts around line 97).
3. Create a new Google Doc (docs.google.com → Blank document).
4. Copy the paper content from the markdown file and paste into the Google Doc.
5. Apply formatting: bold the subheadings, double-space, Times New Roman 12pt.

## What’s included in the HTML/paper

- Full revised paper (~3,400 words)
- MLA-style heading
- Bold, flush-left subheadings for each section
- Interview placeholder with instructions
- Annotated Bibliography template (citations with space for CRAAP annotations)
- Appendix A: AI Declaration placeholder
- Appendix B: Interview placeholder

## After importing

1. **Add your interview** when it’s done (replace the italic placeholder text).
2. **Complete the Annotated Bibliography** annotations for each source.
3. **Fill in Appendix A** (AI Declaration) and **Appendix B** (Interview).
4. **Format:** File → Page setup → set 1" margins, double spacing. Format → Paragraph → Double space. Font: Times New Roman, 12pt.
5. **Page numbers:** Insert → Page numbers → top right, include last name (e.g., Covert 1, Covert 2).

6. **Page breaks (if not preserved from import):** Google Docs often ignores HTML page breaks. To add them manually:
   - Place the cursor at the end of the Conclusion (before "Annotated Bibliography")
   - Go to **Insert → Break → Page break**
   - Repeat before "Appendix A" and before "Appendix B"
